Seja bem-vindo as instruções do software MoniTech,

-Instalando o software:

	Para instalar este software, será necessário que esteja logado como super usuário (root) e que possua conexão com a internet.
	Após verificar esses dois fatores, realize, no shell, o comando:
	"bash install.sh"
	E responda se deseja continuar com a instalação ao não.

-Abrindo o software:

	Após a instalação do programa, verifique o endereço de IP do equipamento onde foi instalado o programa, e o digite no seu browser de preferência (de preferência o browser Mozilla Firefox).

-Acessando o software:

	Ao entrar com o seu IP no browser, aparecerá a tela de Login. Nela, digite como nome de usuário: "admin", e como senha: "admin".
	Pronto, já está com seu programa instalado e pronto para uso, mas antes, leia as recomendações.

-Recomendações: 

	Para começar a utilizar o programa, é aconselhado criar um usuário administrador para o sysadmin (Administrador do Sistema) e excluir o usuário "admin" que, por ser um usuário padrão, outras pessoas que já instalaram o programa terão como entrar em seu sistema de monitoramento.
	Logo após, adicionar todos os equipamentos a serem monitorados no sistema, e depois já iniciar o serviço de monitoramento, assim, o MoniTech já estará pronto para realizar sua função como sistema de monitoramento de rede.

-Removendo o Software:

	Para remover o software é necessário que execute, no shell, o comando:
	"bash remove.sh"
	E responder se deseja continuar com a desinstalação ou não.

-Documentação:

	A documentação do MoniTech encontra-se no diretório Documentacao, dentro deste mesmo diretório, e dentro de Documentacao estará o arquivo relacionado ao melhor entendimento deste software.
	Também há a possibilidade de baixar o arquivo de documentação pelo programa em uma área do menu chamada "Documentação".

-Bugs:

	Informamos aqui alguns bugs que estamos trabalhando para a sua correção.
	1. Ao clicar no botão "Iniciar" na página inicial, ele fica em estado "carregando", por isso, atente-se em clicar logo em seguida no botão "Página Inicial" no menu para voltar a página ao seu estado comum;
	2. O mesmo problema que ocorre com o botão "Iniciar" ocorre com o botão "Reiniciar", por isso, atente-se em clicar logo em seguida no botão "Página Inicial" no menu para a página voltar ao seu estado comum.
	3. Este software funciona apenas em redes com equipamentos com IP fixo, pois caso o IP do equipamento seja alterado, não temos como identificar.
	4. Ao clicar em "Iniciar" mais de uma vez, ele duplica a execução.
